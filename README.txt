CACH CHAY LOCAL
1) Mo terminal trong thu muc nay
2) Chay lenh: node script.js
3) Mo trinh duyet vao: http://127.0.0.1:3000

NEU DUNG GITHUB PAGES
- Push toan bo file len repo
- Bat Settings > Pages
- Chon branch main va folder root
- Mo link Pages de dung

LUU Y
- Du lieu chi luu trong localStorage cua trinh duyet dang dung
- Du lieu khong dong bo giua cac may hay cac trinh duyet
- Export file Excel chuan .xlsx
- Khi xuat Excel, app se hoi co muon them sheet Static hay khong
- Neu chon them sheet Static, app se hoi co muon dat mat khau bao ve sheet Static hay khong
- Sheet Static gom bang xep hang, tong hop nhom diem va bieu do ti le % nhom diem

IMPORT EXCEL
- Bam nut Nhap Excel tren thanh tren cung
- Chon file .xlsx duoc xuat tu chinh web nay
- App se doc cac sheet bang diem, bo qua sheet Static, va hien thi du lieu tren web
- Neu trung hoc sinh trong cung bai lam, hoc ky va ky nang, app se cap nhat dong cu
- File export chi luu diem moi part, khong luu so cau dung. Sau khi import, diem van hien thi dung; neu bam Sua hoc sinh thi can kiem tra lai so cau dung truoc khi cap nhat
